#Задачи на работу со списком.

1. Удалить элемент на позиции n с конца списка.

   Список: 1->2->3->4->5, n = 2; 
   Результат: 1->2->3->5.
   
2. Удалить повторяющиеся элементы в отсортированном списке.

   Список: 1->1->2; 
   Результат: 1->2.
   
   Список: 1->1->2->3->3; 
   Результат: 1->2->3.
